A

Firebase — Google Rekomendasi

Satu ekosistem Google untuk login, database, dan penyimpanan gambar. Tidak perlu server sendiri.

🔐
Auth
Google Sign-In
🗄️
Firestore
Progres + Konten
🖼️
Storage
Upload gambar

Kelebihan

  • Gratis untuk 20–40 siswa
  • Tidak perlu backend/server sendiri
  • Real-time update otomatis
  • CDN-compatible (cocok dengan setup saat ini)
  • Login Google sudah terintegrasi

Kekurangan

  • Perlu akun Google Cloud / Firebase Console
  • Sedikit setup awal (konfigurasi project)
B

Supabase — Open Source

Alternatif Firebase berbasis PostgreSQL. Open source, self-hostable.

🔐
Auth
Google OAuth
🐘
PostgreSQL
Perlu SQL query

Kelebihan

  • Open source, bisa self-host
  • Gratis tier tersedia

Kekurangan

  • Setup lebih kompleks dari Firebase
  • SDK lebih besar, kurang cocok untuk CDN tanpa build
  • Kurang dokumentasi untuk setup no-build
C

Firebase Auth + Google Sheets CMS

Login via Firebase, tapi konten dikelola lewat Google Spreadsheet. Guru cukup edit spreadsheet.

🔐
Auth
Firebase
📊
CMS
Google Sheets

Kelebihan

  • Guru familiar dengan Spreadsheet
  • Tidak perlu UI CMS khusus

Kekurangan

  • Tidak bisa upload gambar lewat Sheets
  • API key terekspos di frontend
  • Tidak real-time (butuh refresh)
  • Progres siswa tetap butuh Firestore
