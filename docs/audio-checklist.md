# Daftar Audio — العربية التفاعلية

Semua file audio disimpan di folder `assets/audio/` dengan struktur berikut.
Format: **MP3**, kualitas minimal **128kbps**, pelafalan fasih (fusha).

---

## Struktur Folder

```
assets/
└── audio/
    ├── hiwar/          ← Dialog percakapan
    ├── mufrodat/       ← Kosakata (kata + contoh kalimat)
    ├── tadribat1/      ← Soal audio Tadribat 1
    └── qawaid/         ← Konjugasi fi'il
```

## Cara Menghubungkan ke App

Setelah semua audio direkam, update field `audio_ref` di `js/data.js`:

```js
// Contoh untuk Mufrodat:
{ ar: 'الطَّبِيب', audio_ref: 'assets/audio/mufrodat/mufrodat_01_at-tabib.mp3', ... }

// Contoh untuk Hiwar:
{ speaker: 'أ', ar: '...', audio_ref: 'assets/audio/hiwar/hiwar_salam_1.mp3', ... }

// Contoh untuk Tadribat 1:
{ type: 'audio', audio_text: 'الطَّبِيب', audio_ref: 'assets/audio/tadribat1/tadribat1_q01.mp3', ... }

// Contoh untuk Qawaid:
{ verb_form: 'ذَهَبَ', audio_text: 'ذَهَبَ', audio_ref: 'assets/audio/qawaid/qawaid_madhi_huwa.mp3', ... }
```
